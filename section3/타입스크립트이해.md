- 타입 호환성 : 어던 타입을 다른 타입으로 취급해도 괜찮은지 판단하는 것

```
let num1: number = 10;
let num2: 10 = 10;
```

&rarr; 일때, num1이 num2일 수는 있지만 num2가 num1가 될 수 없다.

```
num1 = num2 O
num2 = num1 X
```
